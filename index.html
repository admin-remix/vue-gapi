# VueGapi

Google API Client Libraryopen in new window wrapper for Vue.jsopen in new window

# Requirements

Version 2 requires Vue.js v3.xopen in new window.

If you are looking for a Vue.js v2.x compatible version, use Version 1open in new window.

# Installation

npm install --save vue-gapi
1

or

yarn add vue-gapi
1

# Usage

Installing the plugin with gapi.client.initopen in new window configuration parameters:

import Vue from 'vue'
import VueGapi from 'vue-gapi'

const app = Vue.createApp({})

app.use(VueGapi, {
  apiKey: '<YOUR_API_KEY>',
  clientId: '<YOUR_CLIENT_ID>.apps.googleusercontent.com',
  discoveryDocs: ['https://sheets.googleapis.com/$discovery/rest?version=v4'],
  scope: 'https://www.googleapis.com/auth/spreadsheets',
})
