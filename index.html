# VueGapi

Google API Client Library (opens new window) wrapper for Vue.js (opens new window)

# Vue 3

Vue.js v3.x (opens new window) will be supported in v2.0 and can be installed today via vue-gapi@next.

# Installation

npm install --save vue-gapi

or

yarn add vue-gapi

# Usage

Installing the plugin with gapi.client.init (opens new window) configuration parameters:

import Vue from 'vue'
import VueGapi from 'vue-gapi'

Vue.use(VueGapi, {
  apiKey: '<YOUR_API_KEY>',
  clientId: '<YOUR_CLIENT_ID>.apps.googleusercontent.com',
  discoveryDocs: ['https://sheets.googleapis.com/$discovery/rest?version=v4'],
  scope: 'https://www.googleapis.com/auth/spreadsheets',
})

exposes a $gapi member on the Vue instance:

<script>
  export default {
    methods: {
      login() {
        this.$gapi.login().then(({ currentUser, hasGrantedScopes }) => {
          console.log({ currentUser, hasGrantedScopes })
        })
      },
    },
  }
</script>
