# Authentication

Once you have installedopen in new window the plugin, here is a conventional Vue.js v3.x componentopen in new window that displays a login or logout button based on a detected authenticated state.

# Template

<script type="text/x-template" id="login-template">
  <div>
    <div v-if="isSignedIn">
      <button @click="logout()" type="button">Logout</button>
      {{ userName }}
    </div>
    <button
      :disabled="isSignedIn === null"
      @click="login()"
      type="button"
      v-if="!isSignedIn"
    >
      Login
    </button>
  </div>
</script>

# Component

app.component('login', {
  template: '#login-template',
  data() {
    return {
      isSignedIn: null, // (1) Track authenticated state
    }
  },
  created() {
    // (2) Subscribe to authentication status changes
    this.$gapi.listenUserSignIn((isSignedIn) => {
      this.isSignedIn = isSignedIn
    })
  },
  methods: {
    // (3) Expose $gapi methods
    login() {
      this.$gapi.login()
    },
    logout() {
      this.$gapi.logout()
    },
  },
  computed: {
    userName() {
      // (4) Display authenticated user name
      const user = this.$gapi.getUserData()

      if (user) {
        return user.firstName
      }
    },
  },
})

  1. Track authenticated state via an isSignedIn data optionopen in new window property.

  2. Subscribe to authentication status changes via listenUserSignInopen in new window.

  3. Expose loginopen in new window and logoutopen in new window methods.

    Most $gapi methods return a promise. See the GoogleAuthService reference documentationopen in new window for more details.

  4. Display the authenticated user's name via getUserDataopen in new window.
